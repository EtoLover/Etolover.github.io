世界500强企业大数据可视化平台

基于 Hadoop + Spark 的离线分析与可视化展示（课程实验项目）

系统技术架构

  • 数据采集： Fortune Global 500 企业数据（CSV）
  • 分布式存储： Hadoop HDFS
  • 数据仓库： Hive
  • 离线计算： Spark SQL
  • 结果存储： MySQL
  • 缓存层： Redis（Top10 指标缓存）
  • 服务接口： Spring Boot RESTful API
  • 前端展示： ECharts + JavaScript
